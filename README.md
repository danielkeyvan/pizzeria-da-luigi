# Pizzeria da Luigi - Sito Web Moderno e Completo

## 🍕 Descrizione

**Pizzeria da Luigi** è un sito web moderno e completamente responsive per una pizzeria a Treviso. Questo progetto è stato creato come template demo per agenzie e consulenti che desiderano proporre soluzioni web professionali ai loro clienti nel settore horeca.

## 🚀 Installazione e Utilizzo

1. **Clona il repository**
```bash
git clone https://github.com/danielkeyvan/pizzeria-da-luigi.git
cd pizzeria-da-luigi
```

2. **Apri nel browser**
- Semplicemente apri `index.html` nel browser
- O usa un server locale: `python -m http.server 8000`

3. **Personalizzazione**
- Modifica i dati di contatto in HTML
- Cambia i colori in `css/style.css`
- Aggiungi tue foto nella cartella images
- Personalizza il menu con i tuoi piatti
